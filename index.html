What is AH-O₂?

AH-O₂ is a WordPress feature plugin project that's been in the works since August 2013. Our goal is to revitalize and rethink how admin help is presented in the WordPress admin and ultimately included in WordPress 4.0.

What's it do?

AH-O₂ adds two configurable help options to enable/disable help overviews and tooltips.

Help overviews appear at the top of every page and explain what the current page can be used for.

Tooltips appear when hovering over specific items as you navigate the WordPress admin and are intended to give specific information relevant to that task/item about the intended use/functionality.

  • You can hop onto Gitter (Git chat client) and request access or ask for more information.
